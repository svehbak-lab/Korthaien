# Kardex — Prisverktøy

Sammenligner priser fra Mystore med Scryfall-markedspriser for Magic: The Gathering-kort.

## Kom i gang

```bash
npm install
npm run dev
```

Åpne http://localhost:5173

## Produksjon (Vercel/Netlify)

Push til GitHub og koble til Vercel eller Netlify — de bygger automatisk ved hver push.

## Konfigurasjon

Legg inn Mystore URL og API-nøkkel i ⚙ Innstillinger i appen.
Scryfall trenger ingen nøkkel.
